Defining Full-Stack Development

A progressive approach to software development requiring one to have detailed knowledge of the entire Software / Solution Stack. Requiring proficiency in a variety of languages & frameworks.
A Full-Stack Developer can provide much needed experience along with an extended skill-set that can be configured to best fit the requirements of any project.

The term Full-Stack refers to the stack of technologies that serves up the application. One must be skilled in multiple languages, frameworks, SDK / API - Capable of weaving in / out of the Stack while following set standards & best practices. This doesn NOT imply that the developer has mastered everything within the overall SDLC; it DOES imply however, that one has the ability to work with the many layers of the Stack with minimal assistance.
